# 📊 Sales Data Analysis Project

## 📌 Overview

This project focuses on analyzing sales data from a Superstore dataset. It includes data cleaning, transformation, and visualization to extract meaningful insights.

## 🚀 Features

* Data cleaning (handling null values and duplicates)
* Data type correction and preprocessing
* Feature engineering (Year, Month, Profit Margin, etc.)
* Sales and profit analysis
* Data visualization using charts and graphs

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## ⚙️ Workflow

1. Load dataset (Superstore.csv)
2. Clean data (remove duplicates, handle missing values)
3. Convert data types (dates, numeric fields)
4. Create new features (Year, Month, Profit Margin)
5. Perform analysis and visualization

## 📊 Key Insights

* Sales trends over time (Year/Month)
* Profit distribution
* Category-wise and region-wise performance
* Revenue and profit margins

## 📁 Files Included

* main.ipynb – Data cleaning and processing
* analysis.ipynb – Data analysis
* chart.ipynb – Visualizations
* Superstore.csv – Raw dataset

## 👨‍💻 Learning Outcomes

* Data preprocessing techniques
* Exploratory Data Analysis (EDA)
* Visualization skills
* Working with real-world datasets
